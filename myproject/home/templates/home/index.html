Welcome


Welcome to my website!

I made this website in the summer of 2016 to learn about Web dev and with the hope of collating the projects I work on.

You can find my projects in the nav bar on the left.


I built this website myself, with some help from the internet and using the following tools:

  • Back End
    • Apache is running my Ubuntu server
    • Django (Python) is handling requests, hosted on Apache using the mod_wsgi module
  • Front End
    • HTML5Boileplate, for setting up my base Django template
    • jQuery
    • Backbone.js is doing the front end on some of my web apps, like my Physics Engine

Below is an example of one of my projects, a physics engine I've built in Javascript. At the moment, there are particles that rebound according to the laws of momentum and springs that work according to Hooke's Law. The simulation below is just an example, with a Newton cradle (including energy loss), a damped spring and some perfectly elastic particles at the bottom.


It's also interactive.

Click a particle to "kick" it or click and drag to pull it somewhere.

The damped spring only takes so much tension: if you pull it too hard it'll break. The cradle springs however, will hardly extend at all and won't break.
